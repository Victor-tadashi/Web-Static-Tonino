# 🚲 Taller Tonino

Sitio web oficial del **Taller Tonino**, un taller especializado en reparación, mantención y venta de repuestos para bicicletas ubicado en **Quintero, Región de Valparaíso, Chile**.

El sitio fue desarrollado con **HTML5**, **Tailwind CSS** y **JavaScript**, priorizando un diseño moderno, responsivo y orientado a la experiencia del usuario.

---

## 📸 Vista Previa

> Landing Page moderna con:

- Hero con carrusel de imágenes
- Catálogo de servicios
- Carrusel de repuestos
- Información del taller
- Botón de WhatsApp
- Diseño Responsive
- Animaciones al hacer scroll

---

## 🚀 Tecnologías utilizadas

- HTML5
- Tailwind CSS
- JavaScript ES6
- Google Fonts
- Material Symbols
- CSS3

---

📁 Estructura del proyecto

├── css/
│   └── Style.css          # Estilos personalizados adicionales a Tailwind
├── js/
│   ├── Scripts.js         # Lógica e interactividad del sitio
│   └── tailwind-config.js # Configuración personalizada de TailwindCSS
├── index.html              # Página principal
└── README.md
```
